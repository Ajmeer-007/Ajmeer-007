### [Diabetes Prediction API](https://github.com/Ajmeer-007/diabetes-prediction)

> Production-ready REST API serving a trained Random Forest classifier to predict diabetes risk from patient clinical data — designed for integration into healthcare applications.

![Python](https://img.shields.io/badge/Python-000000?style=flat-square&logo=python&logoColor=39FF14)
![Flask](https://img.shields.io/badge/Flask-000000?style=flat-square&logo=flask&logoColor=FFFFFF)
![scikit-learn](https://img.shields.io/badge/scikit--learn-000000?style=flat-square&logo=scikitlearn&logoColor=F7931E)
![REST API](https://img.shields.io/badge/REST_API-000000?style=flat-square&logo=postman&logoColor=FF6C37)

- RESTful POST endpoint accepting structured JSON patient input
- Random Forest classifier trained on clinical diabetes dataset
- Binary classification output: `1 = Diabetes Detected` / `0 = No Diabetes`
- Roadmap: Docker containerization and cloud deployment

### [Document Summarizer & Q&A — RAG Pipeline](https://github.com/Ajmeer-007)

> Retrieval-Augmented Generation system for intelligent document understanding — multi-format ingestion, semantic vector search, and Wikipedia-backed fallback for out-of-document queries.

![Python](https://img.shields.io/badge/Python-000000?style=flat-square&logo=python&logoColor=39FF14)
![FAISS](https://img.shields.io/badge/FAISS-000000?style=flat-square&logo=meta&logoColor=39FF14)
![RAG](https://img.shields.io/badge/RAG-000000?style=flat-square&logo=databricks&logoColor=39FF14)
![HuggingFace](https://img.shields.io/badge/Transformers-000000?style=flat-square&logo=huggingface&logoColor=FFD21E)

- PDF, DOCX, and TXT multi-format document support
- Semantic vector search via FAISS for context-aware retrieval
- Wikipedia-backed fallback for out-of-document queries
- Built with `sentence-transformers`, `faiss-cpu`, and `PyMuPDF`
